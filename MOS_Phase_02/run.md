# MOS Phase 2 - Running Instructions

### 1. Requirements
- C++ Compiler (e.g., `g++` from MinGW/MSYS2)
- `input.txt` (provided in this folder)

### 2. Build
Run the following command in the terminal:
```bash
g++ mos.cpp -o mos2.exe
```

### 3. Run
Execute the compiled program:
```bash
.\mos2.exe
```

### 4. Verification
- Check **`output.txt`** for the simulation results and error messages.
- The console will display a **Memory Dump** (300 words) for each job processed.
- Various test cases are included in `input.txt` to demonstrate error handling (Timeout, Out of Data, etc.).
